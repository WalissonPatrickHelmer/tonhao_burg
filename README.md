# 🍔 Tonhão Burger — Landing Page Comercial

Projeto de website institucional desenvolvido para uma hamburgueria artesanal fictícia localizada em Belo Horizonte/MG, com foco em presença digital, experiência do usuário e conversão de pedidos online.

## 📌 Objetivo do Projeto
Criar uma landing page moderna e responsiva simulando um projeto real para pequenos negócios locais, aplicando boas práticas de desenvolvimento front-end e design orientado à conversão.

## 🚀 Funcionalidades
- ✅ Layout responsivo (Mobile First)
- ✅ Navegação suave (scroll entre seções)
- ✅ Integração direta com WhatsApp para pedidos
- ✅ Exibição de cardápio interativo
- ✅ Seção de avaliações de clientes
- ✅ Integração com Google Maps
- ✅ SEO básico com meta tags
- ✅ Estrutura semântica em HTML5

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Fonts
- Integrações externas (WhatsApp API e Google Maps Embed)

## 🎯 Conceitos Aplicados
- UI/UX Design
- Landing Pages comerciais
- Estrutura semântica
- Responsividade
- Performance básica
- Simulação de projeto real para portfólio

## 📱 Preview
Landing page desenvolvida para representar um negócio real, podendo ser adaptada para clientes reais e pequenos empreendedores.
